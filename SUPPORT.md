# Support

Thanks for using and contributing to MedInternia.

## Questions and Setup Help

For setup questions, first check:

- `README.md`
- `CONTRIBUTING.md`
- `backend/README.md`
- open and closed GitHub issues

If the answer is not already available, open a GitHub issue with:

- the area you are working on, such as frontend, backend, auth, or docs
- your operating system and Node.js version
- the command you ran
- the exact error message or screenshot

## Feature Requests

Open a feature request issue and describe:

- the user problem
- the expected behavior
- any suggested UI or API changes

## Security

For sensitive security concerns, follow `SECURITY.md` instead of opening a public issue.
